Configurez l'accès à votre Gist GitHub pour synchroniser vos données entre appareils.

1 · Token GitHub

Créez un token sur github.com/settings/tokens avec le scope gist uniquement.

2 · Votre Gist de données

✓ Connecté

Créez un nouveau Gist privé (recommandé pour la première fois) ou renseignez l'identifiant d'un Gist existant.
